# Professional Portfolio | Ankit Kunwar

A modern personal portfolio built with **Next.js** and **Tailwind CSS**. This repository showcases the projects and skills of **Ankit Kunwar**, a Computer Engineering student.

## 🚀 Features

- **Modern Architecture**: Built on Next.js for optimal performance.
- **Dynamic Content Management**: Component data is modularized for easy maintainability.
- **Responsive Design**: Fluid layouts ensure a seamless experience across all devices.
- **Integrated Contact Form**: Direct messaging capabilities powered by Web3Forms.

## 🛠 Tech Stack

- **Framework**: Next.js (React)
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## 💻 Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.
